# Cultura de Paz — Prevención de la Violencia Familiar

Este proyecto promueve la **cultura de paz** y el **autocuidado** frente a la violencia en las relaciones familiares o de pareja. Incluye señales de alerta, acciones preventivas y una ruta de apoyo con contactos verificados.

**Tema:** Prevención de la violencia familiar  
**Fuentes:**  
- [Instituto Nacional de las Mujeres](https://www.gob.mx/inmujeres)  
- [UNICEF México](https://www.unicef.org/mexico/)
